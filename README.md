# HR-Analytics-Dashboard-Using-PowerBI

## Project Overview

This project showcases an interactive **HR Analytics Dashboard** developed using Power BI. The dashboard is designed to analyze employee data and provide insights into workforce distribution, gender ratio, departmental performance, and monthly trends. It enables users to explore HR metrics dynamically through filters and slicers, supporting data-driven decision-making.

---

## Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Cleaning & Transformation
* Data Visualization

---

## Dashboard Features

* KPI card displaying total employee headcount
* Gender distribution analysis using a donut chart
* Headcount breakdown by departments and groups
* Monthly headcount trend visualization
* Interactive slicers for Year, Quarter, Gender, and Groups
* Clean and user-friendly dashboard layout

---

## Key Insights

* Majority of employees are concentrated in **Operations** and **Commercial** departments
* Gender distribution indicates an imbalance in workforce composition
* Headcount trends vary across months with noticeable peaks and dips
* Certain groups contribute significantly more to total workforce

## How to Use

1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop**
3. Use slicers and filters to interact with the dashboard
4. Explore insights across different dimensions like time, gender, and departments
